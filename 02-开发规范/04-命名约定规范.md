## 一、文件命名规范

### 1.1 页面文件

| 类型 | 命名规范 | 示例 |
|------|----------|------|
| 页面目录 | 小写 + 连字符 | `pages/index/`, `pages/room-list/` |
| 页面文件 | 小写，与目录同名 | `index.js`, `index.json`, `index.wxml`, `index.wxss` |
| 子页面 | 小写 + 连字符 | `detail.js`, `create.js`, `my-list.js` |

```
pages/
├── index/                    # 首页目录
│   ├── index.js             # 页面逻辑
│   ├── index.json           # 页面配置
│   ├── index.wxml           # 页面结构
│   └── index.wxss           # 页面样式
├── room/
│   ├── list.js              # 会议室列表
│   ├── detail.js            # 会议室详情
│   └── create.js            # 创建预约
└── admin/
    ├── index.js             # 管理后台首页
    ├── login.js             # 管理员登录
    └── room-manage.js       # 会议室管理
```

### 1.2 组件文件

| 类型 | 命名规范 | 示例 |
|------|----------|------|
| 组件目录 | 小写 + 连字符 | `components/custom-button/` |
| 组件文件 | 小写，与目录同名 | `custom-button.js` |

```
components/
├── common/                   # 通用组件
│   ├── custom-button/
│   ├── custom-modal/
│   └── custom-loading/
└── business/                # 业务组件
    ├── room-card/
    ├── booking-calendar/
    └── time-picker/
```

### 1.3 JavaScript 文件

| 类型 | 命名规范 | 示例 |
|------|----------|------|
| 服务类 | 小驼峰 + Service 后缀 | `userService.js`, `bookingService.js` |
| 工具类 | 小驼峰 + Util 后缀 | `dateUtil.js`, `validatorUtil.js` |
| 配置文件 | 小写 + 连字符 | `global-config.js`, `env-config.js` |
| 常量文件 | 大写（仅用于常量定义） | `constants.js` |
| 混入文件 | 小写 + 连字符 | `theme-mixin.js` |

```
services/
├── userService.js           # 用户服务
├── bookingService.js        # 预约服务
└── roomService.js           # 会议室服务

utils/
├── errorHandler.js          # 错误处理
├── dateUtil.js              # 日期工具
└── validatorUtil.js         # 验证工具

config/
├── env.js                   # 环境配置
└── constants.js             # 常量定义
```

### 1.4 云函数文件

| 类型 | 命名规范 | 示例 |
|------|----------|------|
| 云函数目录 | 大驼峰 + Service | `meetingroomService/`, `userService/` |
| 入口文件 | `index.js` | `index.js` |
| 处理器文件 | 小驼峰 | `create.js`, `getList.js` |
| 工具文件 | 小驼峰 | `permission.js`, `response.js` |

```

│   ├── handlers/
│   │   ├── booking/
│   │   │   ├── create.js    # 创建预约
│   │   │   ├── cancel.js    # 取消预约
│   │   │   └── list.js      # 预约列表
│   │   └── room/
│   │       ├── getList.js   # 会议室列表
│   │       └── getDetail.js # 会议室详情

│   │   └── validator.js     # 参数校验
│   ├── index.js             # 入口文件
│   └── package.json
```

## 二、变量命名规范

### 2.1 普通变量

| 类型 | 命名规范 | 示例 |
|------|----------|------|
| 普通变量 | 小驼峰 | `userName`, `meetingRoomList` |
| 常量 | 全大写 + 下划线 | `MAX_RETRY_COUNT`, `API_BASE_URL` |
| 布尔值 | 前缀 is/has/can/should | `isLoading`, `hasPermission`, `canSubmit` |
| 私有变量 | 下划线前缀 | `_internalData`, `_cacheMap` |
| 全局变量 | 大驼峰 | `AppConfig`, `GlobalData` |

```javascript
// 正确示例
const MAX_PAGE_SIZE = 100
const API_BASE_URL = 'https://api.example.com'

let userName = '张三'
let meetingRoomList = []
let isLoading = false
let hasPermission = true
let canSubmit = false

class MyClass {
  constructor() {
    this._privateData = null
    this.publicData = null
  }
}
```

### 2.2 常量定义

```javascript
// config/constants.js
```

## 三、函数命名规范

### 3.1 函数命名规则

| 类型 | 命名规范 | 示例 |
|------|----------|------|
| 获取数据 | get + 名词 | `getUserInfo()`, `getMeetingRoomList()` |
| 设置数据 | set + 名词 | `setUserInfo()`, `setCurrentRoom()` |
| 处理动作 | 动词 + 名词 | `handleSubmit()`, `processBooking()` |
| 异步操作 | 动词 + 名词 + Async | `fetchDataAsync()`, `submitFormAsync()` |
| 事件处理 | handle + 事件名 | `handleTap()`, `handleInputChange()` |
| 初始化 | init + 名词 | `initPage()`, `initUserData()` |
| 检查/判断 | is/has/can + 名词 | `isValidTime()`, `hasConflict()` |
| 更新 | update + 名词 | `updateUserInfo()`, `updateBookingStatus()` |
| 创建 | create + 名词 | `createBooking()`, `createRoom()` |
| 删除 | delete/remove + 名词 | `deleteRoom()`, `removeItem()` |
| 验证 | validate + 名词 | `validateForm()`, `validatePhone()` |
| 转换 | transform/convert + 名词 | `transformData()`, `convertDate()` |
| 加载 | load + 名词 | `loadData()`, `loadMore()` |
| 刷新 | refresh + 名词 | `refreshList()`, `refreshUserInfo()` |

```javascript
// 正确示例
class BookingService {
  // 获取数据
  async getBookingList() { }
  async getBookingDetail(id) { }
  
  // 创建
  async createBooking(data) { }
  
  // 更新
  async updateBookingStatus(id, status) { }
  
  // 删除
  async deleteBooking(id) { }
  
  // 检查
  hasConflict(date, startTime, endTime) { }
  isValidBooking(data) { }
  
  // 验证
  validateBookingData(data) { }
  
  // 处理
  processBookingResult(result) { }
  
  // 转换
  transformBookingData(rawData) { }
}

// 页面中的事件处理
Page({
  // 事件处理
  handleTap(e) { },
  handleInputChange(e) { },
  handleSubmit() { },
  
  // 加载
  async loadData() { },
  async loadMore() { },
  
  // 刷新
  async refreshList() { }
})
```

### 3.2 回调函数命名

## 五、API 命名规范

### 5.1 云函数 API 命名

**统一格式**：`{资源}_{动作}_{操作}`

| 部分 | 说明 | 示例 |
|------|------|------|
| 资源 | 业务模块名 | meetingroom, user, admin, system |
| 动作 | 业务类型 | booking, manage, query, login |
| 操作 | 具体操作 | create, update, delete, getList |

```
# 会议室预约相关
meetingroom_booking_create          # 创建预约
meetingroom_booking_cancel          # 取消预约
meetingroom_booking_getMyList       # 获取我的预约列表
meetingroom_booking_getDetail       # 获取预约详情
meetingroom_booking_approve         # 审批预约
meetingroom_booking_signIn          # 签到

# 会议室管理相关
meetingroom_manage_create           # 创建会议室
meetingroom_manage_update           # 更新会议室
meetingroom_manage_delete           # 删除会议室

# 用户相关
user_login                          # 用户登录
user_getInfo                        # 获取用户信息
user_updateInfo                     # 更新用户信息
user_completeProfile                # 完善用户信息
```

### 5.2 数据库集合命名

| 类型 | 命名规范 | 示例 |
|------|----------|------|
| 集合名 | 小写 + 复数形式 | `users`, `rooms`, `bookings` |
| 字段名 | 小驼峰 | `userName`, `createTime`, `isDeleted` |

```javascript
// 集合名
const COLLECTIONS = {
  USERS: 'users',
  ROOMS: 'rooms',

  CREDIT_RECORDS: 'creditRecords'

// 字段名示例
const userDoc = {
  _id: 'xxx',
  _openid: 'xxx',
  nickName: '张三',
  avatarUrl: 'https://...',
  realName: '张三',
  phone: '13800138000',
  role: 'user',
  createTime: new Date(),
  updateTime: new Date(),
```

## 六、CSS 类名命名规范

### 6.1 BEM 命名规范

采用 BEM（Block Element Modifier）命名规范：

```
.block { }
.block__element { }
.block--modifier { }
```

```css
/* Block: 独立的块 */
.room-card { }

/* Element: 块的子元素 */
.room-card__image { }
.room-card__title { }
.room-card__meta { }

/* Modifier: 块或元素的状态/变体 */
.room-card--featured { }
.room-card--disabled { }
.room-card__title--large { }
```
